# 📝 To-Do List App (Python)

A simple To-Do List application built using Python.

## 🚀 Features
- Add Task
- View Tasks
- Mark Task as Completed
- Delete Task

## 🛠️ Technologies
- Python
- File Handling

## ▶️ How to Run
1. Download project
2. Open in VS Code
3. Run:
   python main.py

## 📌 Note
- Tasks are stored in a file (tasks.txt)
- File will be created automatically
